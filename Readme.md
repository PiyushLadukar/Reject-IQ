# 🚀 RejectIQ

**RejectIQ** is a Streamlit-based customer rejection analytics platform that helps businesses understand why customers reject products or services.

## ✨ Features

- Add customer rejection records
- Interactive dashboard
- Product and city analysis
- Salesperson tracking
- AI-inspired business insights
- Generate and download reports
- Simple CSV-based data storage

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- CSV


## 🚀 Installation

Clone the repository:

```bash
git clone <repository-url>
cd Reject-IQ
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
